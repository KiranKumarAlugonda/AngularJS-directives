Introduction to Angular Directives

What are directives?

Directives allow us to control and manipulate the DOM. Just like jQuery's $(document).ready() used to allow us to attach functionality to DOM objects on traditional websites, Angular's directives allow us to do the same for our async-loaded single-page apps/sites.

Directives can be invoked with custom tag element names, attributes, class names, and comments.

How they work

As soon as a view (set of DOM elements) is loaded, Angular searches through every element and looks for matches to our app's directives. Once it finds a match, it proceeds to:

  1. Compile a 'template' so it can be quickly referenced/used/re-used
  2. Binds/links controller model attributes with the template
  3. Renders the template as browser-friendly HTML code with JavaScript event/data bindings in place
